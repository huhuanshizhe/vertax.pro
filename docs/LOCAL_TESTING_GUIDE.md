### 测试二:真实AI生成测试(需要API密钥)

**目标**:验证完整的AI生成流程

#### 步骤:

1. **获取DashScope API密钥**
   
   a. 访问 https://dashscope.console.aliyun.com/
   b. 注册/登录阿里云账号
   c. 开通DashScope服务
   d. 创建API Key
   e. 复制密钥(格式: `sk-xxxxxxxx...`)

2. **更新环境变量**
   
   编辑 `.env` 文件:
   ```bash
   # 关闭演示模式
   NEXT_PUBLIC_DEMO_MODE="false"
   
   # 配置真实的API密钥
   DASHSCOPE_API_KEY="sk-your-real-api-key-here"
   ```

3. **重启开发服务器**
   ```bash
   # 停止当前服务器(Ctrl+C)
   # 重新启动
   npm run dev
   ```

4. **执行完整测试**
   
   a. 访问 `/customer/social`
   b. 点击"AI 关键词生成"
   c. 配置参数:
      - 目标平台:选择 LinkedIn + X
      - 语气风格:专业
      - 核心词数量:10(减少测试时间)
      - 长尾词数量:5
      - 最小搜索量:50
   
   d. 点击"开始生成"
   e. 等待30-50秒

5. **验证AI生成结果**
   
   检查以下字段是否有真实内容:
   - ✅ 核心关键词(term, category, metrics)
   - ✅ 长尾关键词(term, contentAngle, searchIntent)
   - ✅ 生成的文案内容(非占位符)
   - ✅ Hashtags(与内容相关)
   - ✅ CTA(有实际文本)
   - ✅ 配图提示词(详细描述)

## 🐛 常见问题排查

### 问题1:点击生成后报错"未找到企业能力画像"

**原因**:知识库尚未完成分析,没有企业画像数据

**解决方案**:
1. 访问 `/customer/knowledge` (知识管理页面)
2. 上传企业文档或导入官网页面
3. 点击"分析资产"生成能力画像
4. 等待分析完成后,再回到社媒页面测试

---

### 问题2:API返回空结果或超时

**原因**:API密钥无效或网络问题

**解决方案**:
1. 确认 `DASHSCOPE_API_KEY` 配置正确
2. 检查网络连接(是否能访问阿里云)
3. 查看控制台日志确认错误信息
4. 如果配额用完,考虑充值或切换其他模型

---

### 问题3:生成速度很慢

**优化方案**:
1. 减少测试参数:
   ```json
   {
     "maxCoreKeywords": 5,      // 从30降到5
     "maxLongTailPerCore": 3,   // 从10降到3
     "platforms": ["linkedin"]  // 只选1个平台
   }
   ```
